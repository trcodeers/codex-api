# MockPrep Backend (NestJS + MongoDB)

Production-ready modular backend for mock test preparation platform.

## Setup

1. Copy env file:
   ```bash
   cp .env.example .env
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run in dev:
   ```bash
   npm run start:dev
   ```

## API Base Path

All routes are prefixed with `/api`.

## Seed Data

```bash
npm run seed
```
